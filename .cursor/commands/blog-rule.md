# WordPress投稿記事のHTML/Class自動挿入ルール

このルールは、WordPressの投稿画面で使用するHTMLタグとclassを適切な箇所に挿入するためのガイドラインです。
使用場面と使用禁止を絶対に守りながら書き換えてください

## 使用可能なclass一覧

### 注意点
- 見出し（H2,H3,H4）にclass(マーカーなど)は適用しない
- 二つのstyleを適用しない

### 使用禁止の例
- <span class="marker"><span class="code">array</span></span>
- ↑どちらが一つのみにすること（code優先でOK）
- markerやcode限らずclassが設定されている子セレクタに追加のclass要素はつけないこと。

### 1. マーカー（ハイライト）
重要なキーワードや強調したいテキストに使用します。

```html
<span class="marker">強調したいテキスト</span>
```

**使用場面:**
- 重要なキーワード
- 結論や要点
- 読者に特に注目してほしい部分

**使用禁止:**
- シングルコード、ボックス、リストコンテナ被らないようにする
---

### 2. インラインコード
コード、コマンド、ファイル名、プロパティ名などに使用します。

```html
<span class="code">コードやコマンド</span>
```

**使用場面:**
- 関数名、変数名、プロパティ名
- ファイル名、パス
- 短いコマンド
- 設定値

---

### 3. シングルコード（別スタイル）
インラインコードの別バリエーションです。

```html
<span class="singleCode">コード</span>
```

**使用場面:**
- `.code`と使い分けたい場合
- より控えめなコード表示が必要な場合

**使用禁止:**
- マーカー、ボックス、リストコンテナ取った別スタイルと併用しない/被らない

---

### 4. リンク（アイコン付き）
外部リンクや参考リンクに使用します。

```html
<a href="URL" class="link" target="_blank">リンクテキスト</a>
```

**使用場面:**
- 外部サイトへのリンク
- 参考資料へのリンク
- 関連記事へのリンク

---

### 5. 注意書きボックス（Caution）
警告や注意事項を目立たせるボックスです。

```html
<div class="cautionContainer">
  <span class="cautionLabel">注意</span>
  <p>注意内容をここに記載</p>
</div>
```

**使用場面:**
- 指示があった箇所のみに適応してください。指示がない場合は絶対に適用しないでください

---

### 6. ポイントボックス（Point）
重要なポイントや補足情報を目立たせるボックスです。

```html
<div class="pointContainer">
  <span class="pointLabel">ポイント</span>
  <p>ポイント内容をここに記載</p>
</div>
```

**使用場面:**
- 指示があった箇所のみに適応してください。指示がない場合は絶対に適用しないでください

---

### 7. リストコンテナ
リスト形式のコンテンツを背景付きで表示します。


```html
<ul class="blogList">
<li>リスト内容</li>
<li>リスト内容</li>
</ul>
```

**使用場面:**
- ボックス内で他のスタイル（マーカーやシングルコード）は使わない
- 箇条書きリストのグループ化
- 手順の列挙
- 選択肢の一覧
- 短い説明のリスト

---

### 8. 見出し（自動適用）
h2、h3タグは自動的にスタイルが適用されます。

```html
<h2>大見出し</h2>
<h3>中見出し</h3>
```

---

### 9. コードブロック
複数行のコードには`<pre><code>`を使用します。

```html
<pre><code>
複数行のコード
</code></pre>
```

---

### 10. テーブル
表形式のデータには`<table>`を使用します。

```html
<table>
  <thead>
    <tr>
      <th>ヘッダー1</th>
      <th>ヘッダー2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>データ1</td>
      <td>データ2</td>
    </tr>
  </tbody>
</table>
```

---

## 自動挿入の判断基準

### マーカー（.marker）を挿入する条件
- 「重要」「必須」「注意」などのキーワードが含まれる文
- 結論や要点を述べている部分
- 「〜がポイントです」「〜が大切です」などの表現

### インラインコード（.code）を挿入する条件
- バッククォートで囲まれたテキスト
- 関数名（例: `useState`, `onClick`）
- プロパティ名（例: `width`, `height`）
- ファイル名（例: `App.tsx`, `styles.css`）
- コマンド（例: `npm install`, `git clone`）
- 型名（例: `string`, `number`, `boolean`）

### 注意ボックス（.cautionContainer）を挿入する条件
- 「注意」「警告」「気をつけて」などの表現がある段落
- エラーや問題が発生する可能性について説明している部分
- 「〜しないでください」「〜は避けてください」などの表現

### ポイントボックス（.pointContainer）を挿入する条件
- 「ポイント」「Tips」「コツ」などの表現がある段落
- 「〜すると便利です」「〜がおすすめです」などの表現
- 補足的な説明や追加情報

### リストコンテナ（.blogList）を挿入する条件
- 改行で区切られた短い項目のリスト
- 箇条書きではないが列挙されている内容
- オプションや選択肢の一覧

---

## 適用例

**Before:**
```
useStateは状態管理に使用します。これは重要な概念です。
```

**After:**
```html
<span class="code">useState</span>は状態管理に使用します。これは<span class="marker">重要な概念</span>です。
```

**Before:**
```
注意: この設定を変更すると動作しなくなる可能性があります。
```

**After:**
```html
<div class="cautionContainer">
  <span class="cautionLabel">注意</span>
  <p>この設定を変更すると動作しなくなる可能性があります。</p>
</div>
```
